通过Ubuntu连接服务器的方法
发布于：2017-11-18 15:30:16 标签：/ 超算中心 / sshfs / Ubuntu / 访问：

本教程主要讲述的是如何在Ubuntu16.04操作系统上连接国科智算超算。分3部分：

  • 1）前期准备；
  • 2）连接和挂载超算中心；
  • 3）数据传输测试。

在使用本教程前，首先确认已经收到了超算管理员分配的秘钥。通过Vim或者其他文本编辑工具，可以查看一下这个秘钥。超算中心好比是一个宝藏，这个宝藏的大门上有把锁，而开启这把锁的钥匙，就是管理员给我们的秘钥。
